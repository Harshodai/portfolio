# Harshodai's Portfolio

## Project info

This is a personal portfolio website built with React, Vite, and Tailwind CSS.

## How to Run Locally

You can run this project locally using Docker, even if you don't have Node.js installed.

### Prerequisites

- Docker
- Docker Compose

### Steps

1. **Clone the repository** (if you haven't already):
   ```sh
   git clone <YOUR_GIT_URL>
   cd <YOUR_PROJECT_NAME>
   ```

2. **Run with Docker Compose**:
   ```sh
   docker-compose up --build
   ```

   This will build the Docker image, install dependencies, and start the development server.

3. **Access the Application**:
   Open your browser and navigate to `http://localhost:5173`.

### Development

The Docker setup is configured for hot-reloading. Changes made to files in the `src` directory will automatically reflect in the browser.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
